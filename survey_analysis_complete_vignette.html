Chapter 1: Data Preparation

Foundation for All Psychometric Analyses

Overview

The Data Preparation module serves as the foundation for all subsequent psychometric analyses in this application. It transforms raw survey data into a standardized format while preserving rich contextual information needed for meaningful analysis.

Expected Data Format

Input Data Structure

The application expects survey data in long format with the following essential columns:

Column NameData TypeDescriptionExample Values
respondentIDCharacter/NumericUnique identifier for each survey participant"12345", "RESP_001"
constructNameCharacterHuman-readable name of the construct"Academic Engagement"
constructIDCharacterAlphanumeric identifier for construct grouping"C1", "C2", "C3"
rCodeCharacterUnique identifier for each survey item"Q1", "AE_01"
response_valueNumericThe actual survey response1, 2, 3, 4, 5
qTextCharacterFull text of the survey question"I enjoy learning new things"

Sample Data Creation

Here's how to create sample data that matches the expected format:

# Create sample survey data
library(dplyr)
library(tidyr)

# Define constructs and items
constructs <- data.frame(
  constructID = c("C1", "C1", "C1", "C2", "C2"),
  constructName = c(rep("Academic Engagement", 3), rep("Teacher Support", 2)),
  rCode = c("AE_01", "AE_02", "AE_03", "TS_01", "TS_02"),
  qText = c("I enjoy learning new things", "I feel motivated", 
            "Material is interesting", "Teacher cares", "Teacher helps")
)

Chapter 2: Descriptive Analytics

Comprehensive Exploratory Data Analysis

Overview

The Descriptive Analytics module provides comprehensive exploratory data analysis for survey research. It automatically generates three levels of analysis: question-level statistics, construct-level summaries, and demographic group comparisons.

Group Comparison Framework

Number of GroupsTest UsedPurpose
2 GroupsIndependent t-testCompare means between two groups

Statistical Outputp-value + effect sizeBoth significance and practical importance

Interpretation Guidelines

  • ±0.0 to ±0.2: Minimal practical difference
  • ±0.2 to ±0.5: Small but noteworthy difference
  • ±0.5 to ±0.8: Medium, educationally significant difference
  • ±0.8+: Large, highly significant difference requiring attention
